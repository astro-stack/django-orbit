5 Minutes to Production

Add Django Orbit to your existing project in three simple steps.

settings.py
# 1. Install # pip install django-orbit

# 2. Add to INSTALLED_APPS
INSTALLED_APPS = [ 'orbit', ... ]

# 3. Add middleware (first position)
MIDDLEWARE = [ 'orbit.middleware.OrbitMiddleware', ... ]

# 4. Add URL (urls.py)
urlpatterns = [ path('orbit/', include('orbit.urls')), ]

# Done! Visit /orbit/ to see your dashboard
