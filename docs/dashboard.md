## Overview

The Django Orbit dashboard is available at `/orbit/` and provides:

- **All Events** - Unified feed of all captured events
- **Requests** - HTTP request tracking with headers and body
- **Queries** - SQL queries with timing and N+1 detection
- **Logs** - Python log messages captured automatically
- **Exceptions** - Full tracebacks for all exceptions

## Features

### Real-time Updates
The dashboard automatically refreshes every 3 seconds to show new events.

### Filtering
Click on any sidebar item to filter events by type.

### Detail Panel
Click on any event to see full details in the right panel.
